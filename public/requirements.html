Technical Test Requirements

Summary

You need to populate properties data in certain way in order to complete the test successfully. Bootstrap 3.3.7 and jQuery 2.2.4 JS and CSS files are already loaded through CDNs in index.html. Will highly appreciate using of plain JavaScript, but is not mandatory. Please don't be affraid to use any language or framework documentation.

Project files structure

In the project tree you will find a folder /public from where all HTML, JS and CSS file are served. /public/index.html will be your main HTML file, /public/styles/app.css will be your main CSS file and /public/scripts/app.js will be your main JS file.

Task

Make an AJAX call to http://localhost:3000/api/properties endpoint from where you will receive an array of 12 objects. Each object will have the same structure, just the values will be different for each property. Your task is to dynamically create the HTML for each property and append it to DIV with id of 'properties-list' which is located in /public/index.html file.

For mobile devices the number of properties per row should be 1, for tablets should be 2 and for desktop should be 3. Below you can see an images of how this should look for each device after completion.
